# Prompt Editor

TOX name `base_txt_input`

## Summary

A TouchDesigner component for formatting your prompts for the Google Gemini API. This is a convenience operator to help make merging and managing your prompts easier. You can do all of the same operations with Text and Merge DATs if you prefer.
